# OhMyAICodingToolbox - AI 记忆文件

本文件是 AI 编码工具的项目记忆，包含项目约定和自我学习的知识。

## 项目约定

- 所有 command 使用 `oh.` 前缀
- 使用 `{{MEMORY_PATH}}` 占位符引用工具特定的记忆路径
- 遵循测试驱动开发（TDD）原则
- 规范文档存放在 `specs/{branch-name}/` 目录

## 可用 Commands

| Command | 用途 |
|---------|------|
| `oh.specify` | 创建功能规范（spec.md） |
| `oh.plan` | 创建技术方案（plan.md） |
| `oh.implement` | 执行方案直到测试通过 |

## 自我学习机制

每个开发周期结束后，需要智能更新本记忆文件。

### 记忆更新触发条件

| 条件 | 优先级 |
|------|--------|
| 同一错误出现 2 次以上 | 高 |
| 解决方案需要大量调试时间 | 高 |
| 现有记忆内容不再准确 | 高 |
| 发现意外行为或边界情况 | 中 |
| 新的架构模式或技术选型 | 中 |
| 发现更好的实践方法 | 中 |

### 应该记录的内容

- 花费大量调试时间的错误及解决方案
- 与文档描述不符的行为
- 应该复用的模式
- 项目特定的知识

### 不应该记录的内容

- 通用编程知识
- 不太可能再次发生的一次性修复
- 临时解决方案
